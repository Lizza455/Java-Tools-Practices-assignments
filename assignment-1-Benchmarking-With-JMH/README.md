# Assignment 1 - Benchmarking With JMH

## Overview

This project compares the performance of four sorting algorithms in Java:

* Bubble Sort
* Quick Sort
* Radix Sort
* Arrays.sort()

They were tested on random, sorted, reversed, and almost sorted data using JMH.

---

## Notes

* Bubble Sort: 100,000 elements (too slow for larger inputs)
* Others: 1,000,000 elements
